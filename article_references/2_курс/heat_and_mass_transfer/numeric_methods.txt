одномерное уравнение теплопроводности относится к типу параболических
Имеет два класса: сеточные методы и метод прямых
стр.77-82 (критерий сходимости)

Экономичные алгоритмы
- явная сеточная схема (реализована сейчас)
- явная схема методом прямых, замена только пространственной производной. Производная по времени остается,
далее решается одним из методов решения задачи Коши
- неявная схема 

Комбинации:
- схема Кранка-Никольсона

численные методы:

1) Методы основанные на тейлоровском разложении 
- явный метод Эйлера
- метод эйлера с Пересчетом (метод Хойна)
- метод Коши
- явные методы  Рунге-Кутты (2, 3, 4 порядка)
- явные/неявные методы Аддамса

2) Численные методы решения краевых задач
- Метод стрельбы
- Метод разностной прогонки
- Вариационный метод Ритца (* частный случай метода Галеркина)
- Метод Галеркина
- метод наименьших квадратов (так же частный случай Галеркина)

3) Интерполяция сплайнами
- Слайн (кусочно заданная функция). Наиболее распространенное интерполяционные кубические сплайны
стр. 48
- Метод конечных элементов

4) метод наименьших квадратов

5) численные решения уравнений матфизики
- решение уравнений элептического типа методом сеток
- решение одномерного уравнени теплопроводности (явная схема). Одномерное уравнен теплопроводности - параболическое

интегральные уравения:
1) Численные решения интегральных уравнений
- метод разложения по собственным значениям
- метод замены интеграла конечной суммой
- метод замены ядра
